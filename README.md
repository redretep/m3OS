# m3os - pixel-inspired webos
! this is still beta !
---
#### m3os is a pixel inspired WebOS for school offering a base for unblocked game sites (or android simulation). what it essentially does is scan the subfolder "apps" for folders containing a index.html and a cover image (the apps).
---
#### the look and feel of this webOS should replicate pure android/pixelos and provide a simple structure for unblocked games creators
